# Audotics Technology Stack (Enterprise Edition)

## Frontend Technologies

| Technology | Purpose | Implementation |
|------------|---------|----------------|
| Next.js (v14+) | Framework for SEO, speed, scalability, SSR/SSG | Dynamic music pages, profile, search results |
| Tailwind CSS | CSS utility framework for styling | Buttons, layouts, responsive design |
| Framer Motion | Animation library for complex animations | Transitions between pages, mood-based effects |
| Zustand | State management solution | Global player state, mood states |
| Axios | HTTP client for API handling | Fetching recommendations, user data |
| Socket.IO | Real-time communication engine | Live mood updates, song sync |
| Responsive Design | Mobile and desktop compatibility | Adaptive layout for devices |
| TensorFlow.js | Client-side machine learning | Local mood detection |
| SWR | Data fetching and caching | Auto-refreshing data |
| React Hook Form | Form state management | Login, signup, feedback forms |

## Backend Technologies

| Technology | Purpose | Implementation |
|------------|---------|----------------|
| TypeScript (Node.js) | Core programming language and runtime | Safe, scalable code |
| NestJS | Enterprise-level backend framework | Modular, maintainable API |
| REST + GraphQL | Hybrid API architecture | Fetching song lists, mood data |
| PostgreSQL | Primary relational database | Users, songs, moods, playlists |
| Prisma | Type-safe ORM for database operations | Querying playlists, moods |
| Socket.IO | Real-time communication engine | Live song updates, mood sync |
| Python + TensorFlow | AI microservice implementation | Advanced mood analysis |
| Redis | In-memory caching system | Trending data, session storage |
| JWT + OAuth2 | Authentication and authorization | Secure login, social logins |
| AWS | Cloud infrastructure and deployment | Hosting, file storage, managed DB |
